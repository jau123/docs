**MeiGen** 是一个开源的 MCP (Model Context Protocol) 服务器，赋予 AI 编程助手生成图片的能力。它支持多种生成后端 — 使用 MeiGen 的云端模型、你本地的 ComfyUI 服务器，或任何 OpenAI 兼容 API。

## 什么是 MCP？

Model Context Protocol (MCP) 是一个开放标准，允许 AI 助手使用外部工具。当你将 MeiGen 安装为 MCP 服务器后，你的 AI 助手就能通过自然语言生成图片、搜索精选提示词画廊和增强提示词。

---

## 提供商

MeiGen 支持三种图片生成后端。你可以配置一个或多个 — 插件会自动选择最佳的可用提供商，你也可以明确指定使用哪一个。

<CardGroup cols={3}>
  <Card title="MeiGen Cloud" icon="cloud">
    **托管式多模型 API**。支持 GPT Image 2、Nanobanana Pro/2、Seedream、Midjourney V7/Niji 7、Seedance 等模型。无需 GPU — 只需一个 API Key。
  </Card>
  <Card title="ComfyUI" icon="server">
    **本地 GPU 生成**。在你自己的硬件上运行任何 Stable Diffusion、SDXL 或 Flux 模型。完全控制工作流和参数。
  </Card>
  <Card title="自带 API" icon="plug">
    **任意 OpenAI 兼容 API**。接入 Together AI、Fireworks AI、DeepInfra、硅基流动、OpenAI，或你自己的端点。
  </Card>
</CardGroup>

| 提供商 | 类型 | 费用 | 适用场景 |
| --- | --- | --- | --- |
| MeiGen Cloud | 托管 API | 积分（[定价](/zh/features/models)） | 多模型访问，零配置 |
| ComfyUI | 本地 GPU | 免费（使用你自己的硬件） | 完全控制，自定义工作流，隐私保护 |
| 自带 API | OpenAI 兼容 | 因服务而异 | 已有 API Key，任意供应商 |

**提供商优先级**：当配置了多个提供商时，MeiGen 按以下顺序选择：MeiGen Cloud → ComfyUI → OpenAI 兼容。你可以在每次请求时覆盖此设置。

## 工具

| 工具 | 描述 | 需要提供商？ |
| --- | --- | --- |
| **generate_image** | 使用任意已配置的提供商生成图片 | 是 |
| **enhance_prompt** | 将简短的想法转化为详细、专业的提示词 | 否 |
| **search_gallery** | 搜索 1,500+ 社区精选 AI 艺术提示词 | 否 |
| **get_inspiration** | 获取画廊条目的完整提示词和图片 | 否 |
| **list_models** | 列出所有已配置提供商的可用模型 | 否 |
| **manage_preferences** | 保存默认风格、宽高比和收藏的提示词 | 否 |
| **comfyui_workflow** | 导入、查看、修改和删除 ComfyUI 工作流模板 | 否 |

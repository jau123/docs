**LibreArt** 是 MeiGen 官方的 MCP (Model Context Protocol) 插件。它让你能直接在 Claude Code、Cursor 和 Windsurf 等 AI 编程工具中生成图片 — 无需离开编辑器。

## 什么是 MCP？

Model Context Protocol (MCP) 是一个开放标准，允许 AI 助手使用外部工具。当你将 LibreArt 安装为 MCP 服务器后，你的 AI 助手就能通过自然语言生成图片、搜索画廊和增强提示词。

## 功能

| 工具 | 描述 |
| --- | --- |
| **generate_image** | 使用任意 MeiGen 模型生成图片 |
| **enhance_prompt** | AI 驱动的提示词增强，获得更好的效果 |
| **search_gallery** | 搜索 MeiGen 画廊获取灵感 |
| **list_models** | 浏览可用模型及其能力 |
| **get_inspiration** | 获取随机提示词和图片创作灵感 |

## 免费 vs 付费功能

| 功能 | 需要 API Key？ |
| --- | --- |
| search_gallery | 否 |
| get_inspiration | 否 |
| enhance_prompt | 否 |
| list_models | 否 |
| **generate_image** | **是**（需要 `meigen_sk_` token） |

你可以免费探索画廊、获取灵感和增强提示词。只有实际的图片生成需要带有购买积分的 API Key。

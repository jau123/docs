画廊是 MeiGen 的主页 — 展示来自 X (Twitter) 社区的精选 AI 生成艺术作品。每周更新新内容，涵盖多种 AI 模型的多样化风格、技法和创作方向。

## 浏览画廊

图片以**瀑布流网格**展示，每张卡片按图片原始宽高比显示。

### 分类筛选

使用左侧边栏的分类标签筛选画廊内容：

| 分类 | 内容 |
| --- | --- |
| **产品与品牌** | 产品摄影、Logo、包装、营销素材 |
| **Videos** | 精选 200+ 条来自 X 顶级创作者的 Seedance 视频（跨维度聚合，点击后自动切换到视频数据源） |
| **摄影** | 人像、风景、街头摄影、微距 |
| **插画与 3D** | 数字艺术、概念设计、3D 渲染、角色设计 |
| **美食** | 美食摄影、食谱视觉、餐厅场景 |
| **女孩** | 角色肖像、时尚、生活方式 |

你还可以切换到**最新**视图，按时间顺序查看最新添加的内容。

### 浮动工具栏（桌面端）

屏幕底部的浮动工具栏提供快速访问关键区域的入口：

| 按钮 | 操作 |
| --- | --- |
| **首页** | 返回画廊 |
| **历史记录** | 查看你的生成历史 |
| **收藏夹** | 打开你的收藏集合 |
| **生成视频** | 打开视频生成侧边栏 |
| **生成图片** | 打开图片生成侧边栏 |
| **返回顶部** | 滚动到顶部（滚动后出现） |

## 画廊卡片

每张画廊卡片展示图片及悬停（桌面端）或详情视图（移动端）中可见的上下文信息。

**卡片信息包括：**
- 作者头像、昵称和 @用户名（如适用则显示认证徽章）
- 点赞数和浏览数
- 发布时间（如"2 小时前"、"3 天前"）
- 模型标签，显示使用的 AI 模型

### 悬停操作（桌面端）

当你将鼠标悬停在卡片上时，会出现一个覆盖层，包含三个操作：

<AccordionGroup>
  <Accordion title="使用创意" icon="repeat">
    将图片的原始提示词复制到[生成编辑器](/zh/features/generate)中，并打开生成侧边栏。提示词已就绪，可直接编辑和重新生成。

    此按钮仅在提示词已提取可用时出现。部分较新的帖子可能在提示词处理期间显示"提示词待提取"。
  </Accordion>

  <Accordion title="收藏" icon="layer-group">
    将图片保存到你的个人收藏集合。再次点击可取消收藏。需要登录。

    收藏在所有设备间同步 — 在桌面端保存，在移动端也能访问。
  </Accordion>

  <Accordion title="在 X 上查看" icon="arrow-up-right">
    在新标签页中打开 X (Twitter) 原始帖子，查看完整内容、回复以及创作者的其他作品。
  </Accordion>
</AccordionGroup>

### 拖拽卡片

画廊卡片支持拖拽 — 你可以抓取任意卡片并将其拖到生成侧边栏的不同区域：

- **拖到反推提示词区域**（提示词编辑器上方） — AI 会分析图片并自动生成详细的文字提示词，描述其内容、风格、构图和氛围。
- **拖到参考图片区域** — 图片将作为下次生成的参考。无需下载 — 直接使用图片 URL。

<Tip>
  同样操作也可以通过图片详情对话框里的"用作提示词"和"用作参考"按钮完成。
</Tip>

### 移动端

在移动设备上没有悬停操作。点击卡片会直接打开完整的图片详情视图，所有操作均可在其中使用。

## 图片详情

点击任意画廊卡片打开全屏**图片详情对话框**。这提供了图片的完整视图和所有可用操作。

### 图片查看器

- **完整分辨率** — 图片以原始质量显示
- **多图导航** — 当帖子包含多张图片时，你会看到：
  - 角落的图片计数器（如"2 / 4"）
  - 右侧的缩略图条（桌面端），可快速切换
  - 上一张/下一张箭头按钮
- **下载** — 点击下载按钮（右上角）将完整分辨率图片保存到你的设备
- **关闭** — 点击 X 按钮或按 `Esc` 关闭对话框

### 信息面板

右侧（桌面端）或底部（移动端）显示详细信息：

**作者信息：**
- 头像、昵称、@用户名（附 X 主页链接）
- 如适用则显示认证徽章

**统计数据：**
- 点赞数、浏览数和发布时间
- 模型标签，显示生成该图片的 AI 模型

**提示词区域：**
- 用于生成图片的完整提示词文本（保留原始格式）
- **复制按钮** — 悬停时出现，将提示词复制到剪贴板
- **翻译按钮** — 使用 AI 将提示词翻译为你的语言。再次点击显示原文

**操作按钮：**
- **用作提示词** — 关闭对话框，打开生成侧边栏并填入提示词。可直接编辑和生成。
- **用作参考** — 将当前图片添加为生成侧边栏中的参考图片。适用于风格迁移或构图引导。

<Note>
  如果提示词尚未提取，"用作提示词"按钮会显示为灰色并带有"提示词待提取"标签。稍后再来查看 — 提示词通常在几小时内可用。
</Note>

### 视频内容

画廊也包含视频内容：

- **自主生成的视频**（在 MeiGen 上创建） — 在对话框中自动播放，带有播放控制和循环
- **来自 X 的视频** — 由于跨域限制，无法直接播放。会显示"在 Twitter 上查看"链接，打开原始帖子即可观看视频。

## 搜索

按 `Cmd+K`（Mac）或 `Ctrl+K`（Windows）打开搜索对话框，也可以点击左侧边栏的搜索图标。对话框有两个标签页：

### 帖子标签页

搜索整个画廊的社区艺术作品。

- **混合搜索** — 结合文本匹配和向量（语义）搜索。搜索范围涵盖提示词、描述和视觉内容。
- **筛选器** — 按以下条件缩小结果：
  - **模型** — 筛选特定 AI 模型的图片
  - **分类** — 按内容分类筛选（产品、摄影等）
  - **有提示词** — 仅显示提示词可用的图片
- **空查询** — 未输入任何内容时，搜索默认显示**最新帖子**
- **无限滚动** — 结果随滚动逐步加载
- **实时搜索** — 停止输入后自动触发搜索（300ms 防抖）

### 生成记录标签页

搜索你自己的生成图片。需要登录。

- 按提示词文本搜索你的所有历史生成记录
- 使用键盘导航结果：`↑` / `↓` 方向键移动，`Enter` 打开
- 显示待处理、已完成和失败的生成记录

## 收藏夹

通过侧边栏或浮动工具栏访问你的收藏夹。收藏集合存储两种类型的内容：

- **画廊帖子** — 你从画廊保存的社区艺术作品
- **你的生成作品** — 你生成并收藏的图片

### 管理收藏

- **添加** — 点击任意画廊卡片或生成卡片上的收藏按钮（图层图标）
- **移除** — 再次点击收藏按钮，或在收藏网格中使用移除按钮
- **筛选** — 在全部 / 帖子 / 生成记录标签页间切换，查看特定内容类型
- **同步** — 收藏与你的账户绑定，在所有设备间同步

收藏网格中的卡片也支持悬停操作：**使用创意**（如提示词可用）和**从收藏中移除**。

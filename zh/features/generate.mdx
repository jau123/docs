MeiGen 让你通过统一的界面使用多种 AI 模型生成图片和视频。点击浮动工具栏中的闪光图标（桌面端）或通过导航菜单打开生成侧边栏。

## 模型选择器

在生成侧边栏顶部，模型选择器下拉菜单让你选择要使用的 AI 模型。

每个选项显示：
- **供应商图标** — 模型供应商的视觉标识（Google、ByteDance、Midjourney 等）
- **模型名称** — 模型的显示名称
- **积分消耗** — 该模型每次生成所需的积分

当 MeiGen 新增模型时，未尝试过的模型旁会出现蓝色小圆点标记。选择该模型后圆点消失。

选择不同的模型会自动更新可用的宽高比、分辨率选项和高级设置，以匹配该模型的能力。

## 提示词编辑器

提示词编辑器基于富文本编辑器构建，支持超越纯文本输入的多种功能。

### 编写有效的提示词

- **要具体** — 描述构图、光线、色彩、质感和视角能获得最佳效果
- **简短提示词也可以** — 使用增强按钮自动将简短描述扩展为详细提示词
- **语言** — 大多数模型接受英文提示词。Niji 7 会自动将非英文提示词翻译为英文

### 变量标签

画廊中的某些提示词包含蓝色的 `[placeholder]` 标签 — 例如 `[style]`、`[subject]` 或 `[color palette]`。这些是**可编辑的模板变量**。

点击任意蓝色标签选中它，然后输入你的替换文字。这让你无需重写整个提示词即可轻松创建变体。

### @ 图片引用

当你上传了多张参考图片时，可以在提示词中内联引用特定图片：

1. 在提示词编辑器中输入 `@`
2. 弹出窗口会显示你已上传的参考图片缩略图
3. 选择一张图片插入引用标签（如 `@image1`、`@image2`）
4. 围绕引用编写指令：*"使用 @image1 作为背景风景，并应用 @image2 的配色方案"*

这让你能精确控制每张参考图片如何影响生成结果。

### 翻译按钮

点击提示词编辑器右上角的翻译图标，将你的提示词翻译为英文。当你使用其他语言编写提示词时非常有用，因为大多数模型使用英文输入能产生最佳效果。

### 提示词增强

点击**魔法棒按钮**自动增强你的提示词。AI 会添加构图、光线、氛围和风格等视觉细节。

根据所选模型自动应用两种模式：
- **润色**（非 Niji 7 模型） — 精炼和丰富你的提示词，同时保留原意
- **扩展**（Niji 7） — 将简短描述转化为 Midjourney 优化的语言，带有动漫/插画风格提示

## 参考图片

上传参考图片来引导 AI 的输出。AI 将这些图片作为视觉上下文 — 用于风格匹配、构图引导或主题参考。

- **上传方式** — 拖拽文件到参考区域，或点击浏览
- **支持格式** — JPG、PNG、WebP
- **自动压缩** — 图片在上传前自动压缩至最大 2MB / 2048px
- **缩略图网格** — 上传的图片显示为缩略图，带有 × 按钮用于移除
- **各模型限制** — 参考图片的最大数量因模型而异（0-5 张）。不支持参考图片的模型（如 Z Image Turbo）会完全隐藏此区域。

<Note>
  查看[模型](/zh/features/models)页面了解每个模型的参考图片数量限制。
</Note>

### 从画廊添加参考

你无需先下载图片。直接将任意画廊卡片拖入参考图片区域 — 图片 URL 会自动使用。

<Tip>
  结合画廊参考和 @ 引用实现强大的多参考工作流。例如，拖入两张画廊卡片作为参考，然后编写："结合 @image1 的构图和 @image2 的配色方案。"
</Tip>

### 参考类型（仅 Niji 7）

使用 Midjourney Niji 7 并添加参考图片时，会出现下拉菜单选择参考的解读方式：

| 类型 | 描述 |
| --- | --- |
| **内容参考** | AI 将图片作为主题素材使用 — 尝试重现类似内容。通过 **Image Weight** (iw) 参数控制影响程度（0-2）。 |
| **风格参考** | AI 提取视觉风格（色彩、氛围、技法），不复制内容。通过 **Style Weight** (sw, 0-1000) 控制。 |

### 何时使用参考图片

某些提示词在配合参考图片时效果最佳，甚至必须搭配参考图才有意义。平台会自动检测提示词中的关键词（如"参考图"、"uploaded image"、"reference image"），并在未上传参考图时显示提醒。

**常见需要参考图的提示词模式：**

| 提示词模式 | 为何需要参考图 |
| --- | --- |
| "将这张参考图转换为..." | AI 需要一张源图进行转换 |
| "基于上传的图片，创建..." | 明确引用了上传的图片 |
| "保持 @image1 的构图，但改变..." | 使用 @ 引用语法引用上传图 |
| "重绘/重新想象参考照片..." | 需要源图来进行重绘 |

<Tip>
  如果你的提示词涉及变换、重绘或引用某张具体图片，请务必先上传该图片作为参考。否则 AI 生成的结果将与你的意图无关。
</Tip>

## 生成选项

### 数量

选择每次请求生成的图片数量（1-4 张）。

- **免费用户** — 每次请求限 1 张图片。+ 按钮上会显示锁定图标。
- **付费用户** — 每次请求最多可并行生成 4 张图片。

### 宽高比

从一系列宽高比中选择。选择器会自动筛选，仅显示当前模型支持的比例。

| 比例 | 方向 | 常见用途 |
| --- | --- | --- |
| 1:1 | 正方形 | 社交媒体头像、图标 |
| 3:4, 4:5, 2:3 | 竖向 | 手机壁纸、海报 |
| 9:16 | 高竖向 | Stories、Reels |
| 1:4, 1:8 | 超高竖向 | 滚动横幅（仅 Nanobanana 2） |
| 4:3, 3:2 | 横向 | 桌面壁纸 |
| 16:9, 21:9 | 宽幅 | 电影画面、超宽屏 |
| 4:1, 8:1 | 超宽幅 | 全景横幅（仅 Nanobanana 2） |

<Warning>
  并非所有模型都支持所有比例。选择器会自动筛选，仅显示当前模型支持的比例。查看[模型](/zh/features/models)页面了解每个模型支持的比例。
</Warning>

### 分辨率

在可用时选择输出分辨率：

| 分辨率 | 可用性 |
| --- | --- |
| **2K** | 所有用户可用（默认） |
| **3K** | 仅部分模型可用（如 Seedream 5.0 Lite） |
| **4K** | 仅付费用户可用 — 免费用户会看到锁定图标 |

## Niji 7 高级选项

Niji 7 是 Midjourney 专注于动漫和插画的模型 — 擅长动漫风格、角色设计和风格化插画。每次生成返回 4 张候选图，可在图片详情对话框中查看全部。

使用 Midjourney Niji 7 时，模型选择器旁会出现齿轮图标。点击打开带有高级参数的弹出面板：

| 参数 | 范围 | 默认值 | 描述 |
| --- | --- | --- | --- |
| **Stylize** | 0-1000 | 150 | 控制 Midjourney 美学风格的应用强度。值越低越写实，值越高越艺术。 |
| **Chaos** | 0-100 | 0 | 增加多次运行间的变化。值越高结果越出乎意料。 |
| **Weird** | 0-3000 | 0 | 引入不寻常的实验性质感。 |
| **Raw Mode** | 开/关 | 关 | 关闭时充分发挥 Niji 7 的动漫风格优化；开启则生成更原始、更少后处理感的图片。建议保持关闭以获得最佳动漫效果。 |
| **Image Weight (iw)** | 0-2 | 1 | 内容参考图片的影响程度。仅在设置了内容参考时可见。 |
| **Style Reference (sref)** | URL/文本 | — | 提供一个你想模仿其风格的图片 URL。当参考图片设为风格模式时禁用。 |
| **Style Weight (sw)** | 0-1000 | 100 | 风格参考的应用强度。仅在有 sref 或风格参考图片时可见。 |
| **Style Version (sv)** | 1-4 | 4 | 使用哪个风格算法版本。 |

当任何参数被修改为非默认值时，齿轮图标会变为蓝色。

## 反向提示词

看到喜欢的图片但不知道如何描述？**反向提示词**使用 AI 视觉分析图片，生成捕捉其内容、风格和构图的详细文字提示词。

### 使用方法

1. **图片详情对话框** — 打开任意图片的详情视图，点击**反向提示词**按钮。AI 分析图片并生成描述性提示词。
2. **拖放** — 将任意外部图片（或画廊卡片）拖到生成侧边栏提示词编辑器上方的**反向提示词投放区**。

### 生成内容

AI 检查图片并生成描述以下内容的提示词：

- 主题和场景构图
- 艺术风格和技法（写实、插画、水彩等）
- 光线、配色方案和氛围
- 相机角度和视角

你可以直接使用生成的提示词，也可以编辑它来创建原始图片的变体。

<Tip>
  反向提示词非常适合作为起点。试试将其与[提示词增强](/zh/features/prompt-enhance)结合 — 先对喜欢的图片使用反向提示词，然后增强结果以获得更多细节。
</Tip>

## 视频生成

MeiGen 还支持 AI 视频生成。点击浮动工具栏中的**场记板图标**打开视频生成侧边栏。

### 帧图片

上传最多 2 张帧图片来控制视频的构图：

- **首帧** — 设置视频的起始构图
- **末帧** — 设置视频的结束构图

两者都是可选的。你也可以直接拖入画廊卡片作为帧图片。

### 视频选项

| 选项 | 可选值 | 描述 |
| --- | --- | --- |
| **宽高比** | 16:9, 9:16 | 横版或竖版视频 |
| **分辨率** | 720p | 输出分辨率（因模型而异） |
| **风格** | 自动 / 写实 / 动漫 / 电影 | 视觉风格预设 |
| **速度** | 自动 / 慢速 / 快速 | 播放速度控制 |

<Note>
  Veo 3.1 会为你的视频自动生成 AI 音频 — 无需单独的音频输入。查看[模型](/zh/features/models)页面了解完整的视频模型规格。
</Note>

### 将图片转为视频

生成图片后，将鼠标悬停在完成的卡片上，点击**动画化**按钮（场记板图标）。这会将生成的图片发送到视频侧边栏作为首帧，让你轻松将静态图片转为短视频片段。

## 生成历史

提交生成请求后，你的图片会以卡片形式出现在生成历史面板中（可通过浮动工具栏的历史记录按钮或侧边栏访问）。

### 处理中状态

图片生成过程中，卡片显示：

- **动画渐变背景**，带有微妙的脉冲效果
- **环形进度条** — 使用指数衰减曲线，开始时快速推进，接近预计完成时间时减慢
- **预计时间** — 视频生成会显示时间估计（如"2-5 分钟"，720p）

生成时间因模型而异：

| 模型 | 典型时间 |
| --- | --- |
| Z Image Turbo | 约 5 秒 |
| Nanobanana 2, Seedream | 约 15 秒 |
| Nanobanana Pro | 约 15 秒 |
| GPT image1.5 | 约 20 秒 |
| Niji 7 | 约 60 秒 |
| Veo 3.1（视频） | 2-6 分钟 |

### 失败状态

如果生成失败，卡片显示：

- 用户友好的错误信息，说明出了什么问题
- **"积分已退还"**提示 — 生成失败时积分会自动退还
- **重试按钮** — 点击以使用相同的提示词和设置重新提交请求

### 完成状态

将鼠标悬停在已完成的生成卡片上，显示操作按钮：

| 操作 | 描述 |
| --- | --- |
| **编辑** | 将生成的图片添加为新一次生成的参考 |
| **抠图** | 移除背景，创建透明 PNG |
| **动画化** | 将静态图片转为视频（图生视频） |
| **删除** | 从你的历史记录中移除该卡片 |
| **使用创意** | 将提示词复制回编辑器以供复用 |
| **收藏** | 保存到你的收藏集合 |
| **下载** | 将图片保存到你的设备 |
| **分享到 X** | 打开预填好的推文，附带你生成的图片 |

卡片左下角还会显示宽高比和分辨率标签。

## 背景移除（抠图）

生成图片后，将鼠标悬停在上面并点击**抠图**按钮来移除背景。这会创建一个透明 PNG — 适用于 Logo、贴纸、产品照片或合成。

抠图过程会在你的生成历史中创建一张新卡片。原始图片保持不变。

## 积分和计费

### 积分如何运作

MeiGen 使用双积分系统：

- **每日积分** — 每 24 小时刷新的免费积分。所有用户可用。
- **购买积分** — 你购买的积分，永不过期。每日积分用完后使用。

通过网站生成时，**优先使用每日积分**，然后使用购买积分。积分消耗会在你点击之前显示在生成按钮上。

你可以将鼠标悬停在左侧边栏底部的积分卡片上查看积分余额。悬停卡片会显示每日和购买积分的分项明细，带有双色进度条。

### 免费 vs 付费

| 功能 | 免费 | 付费 |
| --- | --- | --- |
| 每日积分 | 有（每 24 小时重置） | 有 + 购买积分 |
| 4K 分辨率 | 锁定 | 可用 |
| 每次请求图片数 | 1 | 最多 4 |
| API 访问 | 不可用 | 可用（仅限购买积分） |

## 常见问题

<AccordionGroup>
  <Accordion title="为什么我的生成时间比预期长？" icon="clock">
    生成时间因模型和服务器负载而异。Niji 7 通常需要约 60 秒，Veo 3.1 视频生成需要 2-6 分钟。进度条使用的是估计值 — 高峰时段实际时间可能有所不同。
  </Accordion>

  <Accordion title="生成失败了 — 我的积分怎么办？" icon="coins">
    生成失败时积分会自动退还。退款会立即反映在你的余额中。你可以使用重试按钮再次尝试。
  </Accordion>

  <Accordion title="我能再次生成同一张图片吗？" icon="repeat">
    在失败的卡片上使用**重试**按钮，或在已完成的卡片上点击**使用创意**将提示词复制回编辑器。请注意，AI 生成是非确定性的 — 即使使用相同的提示词，结果也会有所不同。
  </Accordion>

  <Accordion title="抠图和动画化有什么区别？" icon="scissors">
    **抠图**移除静态图片的背景，创建透明 PNG。**动画化**使用视频生成模型将静态图片转为短视频片段。
  </Accordion>

  <Accordion title="为什么分辨率选择器显示锁定图标？" icon="lock">
    4K 分辨率仅付费用户可用。免费用户可以 2K 分辨率生成。升级你的账户以解锁更高分辨率。
  </Accordion>
</AccordionGroup>

MeiGen 让你通过统一的界面使用多种 AI 模型生成图片和视频。通过浮动工具栏（桌面端）或导航菜单打开生成侧边栏。

## 模型选择器

在生成侧边栏顶部，模型选择器下拉菜单让你选择要使用的 AI 模型。

每个选项显示：
- **供应商图标** — 模型供应商的视觉标识（Google、ByteDance、Midjourney 等）
- **模型名称** — 模型的显示名称
- **积分消耗** — 该模型每次生成所需的积分

选择不同的模型会自动更新可用的宽高比、分辨率选项和高级设置，以匹配该模型的能力。

## 提示词编辑器

### 编写有效的提示词

- **要具体** — 描述构图、光线、色彩、质感和视角能获得最佳效果
- **简短提示词也可以** — 使用增强按钮自动将简短描述扩展为详细提示词
- **语言** — 大多数模型接受英文提示词。Niji 7 会自动将非英文提示词翻译为英文

### 变量标签

画廊中的某些提示词包含蓝色的 `[placeholder]` 标签 — 例如 `[style]`、`[subject]` 或 `[color palette]`。这些是**可编辑的模板变量**。

点击任意蓝色标签选中它，然后输入你的替换文字。这让你无需重写整个提示词即可轻松创建变体。

### @ 图片引用

当你上传了多张参考图片时，可以在提示词中内联引用特定图片：

1. 在提示词编辑器中输入 `@`
2. 弹出窗口会显示你已上传的参考图片缩略图
3. 选择一张图片插入引用标签（如 `@image1`、`@image2`）
4. 围绕引用编写指令：*"使用 @image1 作为背景风景，并应用 @image2 的配色方案"*

这让你能精确控制每张参考图片如何影响生成结果。

### # 颜色选择

当你想精确描述某个物体的颜色时，可以从色板中快速选取：

1. 在提示词编辑器中输入 `#`
2. 弹出色板，包含常用色和按色相分组的精选色
3. 点击任意色块插入颜色标签（如 `#dc143c`）
4. 围绕颜色编写指令：*"一件 T 恤，颜色是 `#dc143c`，搭配 `#f5f5dc` 的裤子"*

颜色以 hex 值的形式提交给模型，比单纯描述"深红"、"米色"更精确。

### 翻译按钮

点击提示词编辑器右上角的翻译图标，将你的提示词翻译为英文。当你使用其他语言编写提示词时非常有用，因为大多数模型使用英文输入能产生最佳效果。

### 提示词增强

通过**润色提示词**按钮根据当前模型自动增强提示词：

- **润色模式**（非 Niji 7） — 保留原意，补充构图、光线、材质等视觉细节
- **扩展模式**（Niji 7） — 扩展为 Midjourney 优化的语言，带动漫/插画风格提示；非英文 prompt 会自动翻译

<CodeGroup>
```text 输入
a cat sitting on a windowsill
```

```text 润色后
A fluffy orange tabby cat sitting gracefully on a sunlit windowsill,
warm golden hour light streaming through sheer curtains, soft bokeh
of a garden visible outside, cozy interior with houseplants, photorealistic
style with shallow depth of field
```
</CodeGroup>

<Tip>
  增强最适合 30 词以下的简短 prompt。已有详细视觉描述的 prompt 可跳过 —— AI 可能过度处理。
</Tip>

| 快捷键 | 操作 |
| --- | --- |
| `Cmd+Enter` / `Ctrl+Enter` | 触发生成 |

## 参考图片

上传参考图片来引导 AI 的输出。AI 将这些图片作为视觉上下文 — 用于风格匹配、构图引导或主题参考。

- **上传方式** — 拖拽文件到参考区域，或点击浏览
- **支持格式** — JPG、PNG、WebP
- **自动压缩** — 图片在上传前自动压缩至最大 2MB / 2048px
- **缩略图网格** — 上传的图片显示为缩略图，带有 × 按钮用于移除
- **各模型限制** — 参考图片的最大数量因模型而异（0-5 张）。不支持参考图片的模型（如 Z Image Turbo）会完全隐藏此区域。

<Note>
  查看[模型](/zh/features/models)页面了解每个模型的参考图片数量限制。
</Note>

### 从画廊添加参考

你无需先下载图片。直接将任意画廊卡片拖入参考图片区域 — 图片 URL 会自动使用。

<Tip>
  结合画廊参考和 @ 引用实现强大的多参考工作流。例如，拖入两张画廊卡片作为参考，然后编写："结合 @image1 的构图和 @image2 的配色方案。"
</Tip>

### 参考类型（仅 Niji 7）

使用 Midjourney Niji 7 并添加参考图片时，会出现下拉菜单选择参考的解读方式：

| 类型 | 描述 |
| --- | --- |
| **内容参考** | AI 将图片作为主题素材使用 — 尝试重现类似内容。通过 **Image Weight** (iw) 参数控制影响程度（0-2）。 |
| **风格参考** | AI 提取视觉风格（色彩、氛围、技法），不复制内容。通过 **Style Weight** (sw, 0-1000) 控制。 |

### 何时使用参考图片

某些提示词在配合参考图片时效果最佳，甚至必须搭配参考图才有意义。平台会自动检测提示词中的关键词（如"参考图"、"uploaded image"、"reference image"），并在未上传参考图时显示提醒。

**常见需要参考图的提示词模式：**

| 提示词模式 | 为何需要参考图 |
| --- | --- |
| "将这张参考图转换为..." | AI 需要一张源图进行转换 |
| "基于上传的图片，创建..." | 明确引用了上传的图片 |
| "保持 @image1 的构图，但改变..." | 使用 @ 引用语法引用上传图 |
| "重绘/重新想象参考照片..." | 需要源图来进行重绘 |

<Tip>
  如果你的提示词涉及变换、重绘或引用某张具体图片，请务必先上传该图片作为参考。否则 AI 生成的结果将与你的意图无关。
</Tip>

## 生成选项

### 数量

选择每次请求生成的图片数量（1-4 张）。

- **免费用户** — 每次请求限 1 张图片。+ 按钮上会显示锁定图标。
- **付费用户** — 每次请求最多可并行生成 4 张图片。

### 宽高比

默认为 **Auto** — 无需手动选择，系统会挑一个合适的比例。想要精确控制时从下拉中选具体比例即可。

| 比例 | 方向 | 常见用途 |
| --- | --- | --- |
| **Auto**（默认）| — | 免手动 |
| 1:1 | 正方形 | 社交媒体头像、图标 |
| 3:4, 4:5, 2:3 | 竖向 | 手机壁纸、海报 |
| 9:16 | 高竖向 | Stories、Reels |
| 1:4, 1:8 | 超高竖向 | 滚动横幅（仅 Nanobanana 2） |
| 4:3, 3:2 | 横向 | 桌面壁纸 |
| 16:9, 21:9 | 宽幅 | 电影画面、超宽屏 |
| 4:1, 8:1 | 超宽幅 | 全景横幅（仅 Nanobanana 2） |

<Warning>
  并非所有模型都支持所有比例。选择器会自动筛选，仅显示当前模型支持的比例。查看[模型](/zh/features/models)页面了解每个模型支持的比例。
</Warning>

### 分辨率

在可用时选择输出分辨率：

| 分辨率 | 可用性 |
| --- | --- |
| **2K** | 所有用户可用（默认） |
| **3K** | 仅部分模型可用（如 Seedream 5.0 Lite） |
| **4K** | 仅付费用户可用 — 免费用户会看到锁定图标 |

## Niji 7 高级选项

使用 Midjourney Niji 7 时，可通过**高级选项**按钮自定义更多生图参数。**大部分用户保持默认即可**。

<Accordion title="参数详情（Stylize / Chaos / Weird / Raw / iw / sref / sw / sv）" icon="sliders">

| 参数 | 范围 | 默认值 | 描述 |
| --- | --- | --- | --- |
| **Stylize** | 0-1000 | 100 | 控制 Midjourney 美学风格的应用强度。值越低越写实，值越高越艺术。 |
| **Chaos** | 0-100 | 0 | 增加多次运行间的变化。值越高结果越出乎意料。 |
| **Weird** | 0-3000 | 0 | 引入不寻常的实验性质感。 |
| **Raw Mode** | 开/关 | 关 | 关闭时充分发挥 Niji 7 的动漫风格优化；开启则生成更原始、更少后处理感的图片。建议保持关闭以获得最佳动漫效果。 |
| **Image Weight (iw)** | 0-2 | 1 | 内容参考图片的影响程度。仅在设置了内容参考时可见。 |
| **Style Reference (sref)** | URL/文本 | — | 提供一个你想模仿其风格的图片 URL。当参考图片设为风格模式时禁用。 |
| **Style Weight (sw)** | 0-1000 | 100 | 风格参考的应用强度。仅在有 sref 或风格参考图片时可见。 |
| **Style Version (sv)** | 1-4 | 4 | 风格参考版本。默认 4。 |

</Accordion>

## 反推提示词

看到喜欢的图片但不知道如何描述？**反推提示词**使用 AI 视觉分析图片，生成捕捉其内容、风格和构图的详细文字提示词。

### 使用方法

1. **图片详情对话框** — 打开任意图片的详情视图，点击**反推提示词**按钮。AI 分析图片并生成描述性提示词。
2. **拖放** — 将任意外部图片（或画廊卡片）拖到生成侧边栏提示词编辑器上方的**反推提示词投放区**。

### 生成内容

AI 检查图片并生成描述以下内容的提示词：

- 主题和场景构图
- 艺术风格和技法（写实、插画、水彩等）
- 光线、配色方案和氛围
- 相机角度和视角

你可以直接使用生成的提示词，也可以编辑它来创建原始图片的变体。

<Tip>
  反推提示词非常适合作为起点。可以与[提示词增强](#提示词增强)结合 — 先对喜欢的图片做反推提示词，然后增强结果以获得更多细节。
</Tip>

## 视频生成

MeiGen 支持两个视频模型：**Seedance 2.0**（按秒计费，4-15 秒可调）和 **Veo 3.1**（固定 8 秒，自带音频与风格预设）。通过浮动工具栏的**生成视频**按钮打开视频侧边栏。

### 帧图片

上传最多 2 张帧图片来控制视频的构图：

- **首帧** — 设置视频的起始构图
- **末帧** — 设置视频的结束构图

两者都是可选的。你也可以直接拖入画廊卡片作为帧图片。

### Seedance 2.0

| 选项 | 可选值 | 描述 |
| --- | --- | --- |
| **宽高比** | adaptive, 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 | 7 种比例。`adaptive` 会根据参考图/参考视频自动适配尺寸 |
| **分辨率** | 480p, 720p（默认 480p） | 480p 更省积分 |
| **视频时长** | 4-15 秒（默认 5 秒） | 滑块或输入框选择 |
| **音频** | 自动生成 | 不可关 |
| **参考视频** | 支持 | 用于"视频续写"，参考视频会拼接在生成内容之前 |

**计费**：按秒计费，详细公式（含参考视频时的最低计费表）见[模型](/zh/features/models)。提交前侧边栏会显示精确的总积分。

**三种模式**：
- 文生视频：只写 prompt
- 图生视频：上传 1-2 张帧图片控制首帧/末帧
- 视频续写：上传参考视频（详见下方[续写视频](#续写视频)章节）

<Warning>
  **带参考视频时总积分受参考视频时长影响**：后端会按"参考视频时长 + 你选的 duration"作为计费秒数（不低于最低计费秒数）。参考视频越长，总积分越高。提交前侧边栏会显示精确的总积分，请确认后再点生成。
</Warning>

### Veo 3.1 — 固定 8 秒带音频

| 选项 | 可选值 | 描述 |
| --- | --- | --- |
| **宽高比** | 16:9, 9:16 | 仅横版或竖版 |
| **分辨率** | 720p | 固定 |
| **视频时长** | 8 秒（固定） | 不可调 |
| **风格** | 自动 / 写实 / 动漫 / 电影 | 视觉风格预设 |
| **速度** | 自动 / 慢速 / 快速 | 播放速度控制 |
| **音频** | 自动生成 | 不可关 |

**计费**：每个视频固定 20 积分。

<Note>
  两个视频模型都自动生成 AI 音频，无需单独输入。完整规格、计费公式与对比见[模型](/zh/features/models)页面。
</Note>

### 将图片转为视频

生成图片后，将鼠标悬停在完成的卡片上，点击**生成视频**按钮。这会将生成的图片发送到视频侧边栏作为首帧，让你轻松将静态图片转为短视频片段。

### 续写视频

仅 Seedance 2.0 支持。可以用一段已有视频当起点，再生成一段新内容拼在后面。

#### 三种入口

1. **从视频卡片**：鼠标悬停在任意完成的视频卡片上，点左上角的**续写**按钮 — 该视频会自动载入 Seedance 侧边栏作为参考视频
2. **拖视频到参考图模块**：拖一段视频（本地或画廊）到参考图上传区，会弹窗询问"用作续写参考？"
3. **直接上传**：在 Seedance 侧边栏的"Extend or reference video"上传区选择本地视频

#### 工作流

1. 触发上述任一入口，参考视频会出现在侧边栏预览区
2. Prompt 框会自动预填 `"Extend this video with the following plot: "` — 在后面补你想要的剧情
3. 选择分辨率和新视频时长（4-15 秒）
4. 点生成

#### 关键行为

<Warning>
  **Prompt 必须明确说"extend / 续写"**：如果 prompt 里只描述场景不提续写意图，模型默认会把视频**当参考**而非拼接。侧边栏自动预填的 `Extend this video with the following plot: ` 前缀就是为了避免这个坑，请保留它再补内容。
</Warning>

<Note>
  **输出 = 参考视频 + 新内容拼接**：生成结果是一段连续视频，前半段是你的参考视频原样保留，后半段是 AI 按 prompt 续写的新内容。总时长 ≈ 参考时长 + 你设的 duration。
</Note>

#### 计费

续写模式单价更低（480p 8 积分/秒、720p 16 积分/秒），但**计费秒数会包含参考视频时长**（不低于最低计费秒数）。完整公式、最低计费表和示例见[模型 → Seedance 2.0](/zh/features/models)。

典型成本：
- 续写 5 秒一段 3 秒视频：**72 积分**（480p）
- 续写 10 秒一段 5 秒视频：**272 积分**（720p）

## 生成历史

提交生成请求后，你的图片会以卡片形式出现在生成历史面板中（可通过浮动工具栏的历史记录按钮或侧边栏访问）。

### 处理中状态

生成中的卡片会实时显示进度；视频生成还会标注预计完成时间（如 `2-5 分钟`）。

不同模型的典型生成时间：

| 模型 | 典型时间 |
| --- | --- |
| Z Image Turbo | 约 5 秒 |
| Nanobanana 2, Seedream | 约 15 秒 |
| Nanobanana Pro | 约 15 秒 |
| GPT image1.5 | 约 20 秒 |
| Niji 7 | 约 60 秒 |
| Seedance 2.0（视频） | 约 1-3 分钟（按时长/分辨率而异）|
| Veo 3.1（视频） | 2-6 分钟 |

### 失败状态

如果生成失败，卡片显示：

- 错误原因说明
- **"积分已退还"**提示 — 生成失败时积分会自动退还
- **重试按钮** — 点击以使用相同的提示词和设置重新提交请求

### 完成状态

将鼠标悬停在已完成的生成卡片上，显示操作按钮：

| 操作 | 描述 |
| --- | --- |
| **编辑** | 将生成的图片添加为新一次生成的参考 |
| **抠图** | 移除背景，创建透明 PNG（适用于 Logo、贴纸、产品合成）。会创建一张新卡片，原图保留 |
| **生成视频** | 将静态图片转为视频（图生视频） |
| **删除** | 从你的历史记录中移除该卡片 |
| **使用创意** | 将提示词复制回编辑器以供复用 |
| **收藏** | 保存到你的收藏集合 |
| **下载** | 将图片保存到你的设备 |
| **分享到 X** | 打开预填好的推文，附带你生成的图片 |

卡片左下角还会显示宽高比和分辨率标签。

## 积分和计费

MeiGen 采用双积分制：

- **每日积分**：每 24 小时刷新，用于**基础模型**（Z Image Turbo、GPT image 1.5）。
- **购买积分**：通过充值获得，永不过期，可用于所有模型。

生成高级模型（Nanobanana、Seedream、Midjourney、Seedance、Veo 等）需要购买积分。

**API Token 调用**只使用购买积分，不论调用哪个模型；每日积分在 API 路径永远不参与扣费。

积分消耗会在提交前显示在生成按钮上；将鼠标悬停在左侧边栏底部的积分卡片可查看余额明细。

### 免费 vs 付费

| 功能 | 免费用户 | 付费用户 |
| --- | --- | --- |
| 4K 分辨率 | 锁定 | 可用 |
| 每次请求图片数 | 1 | 最多 4 |
| API 访问 | 不可用 | 可用（仅购买积分） |

MeiGen 支持多种 AI 模型用于图片和视频生成。先看下方对比表决定用哪个，再翻到具体章节查看参数细节。

## 快速对比

| 模型 | 积分 | 速度 | 参考图片 | 最适用场景 |
| --- | --- | --- | --- | --- |
| **GPT Image 2.0**（默认）| 2–80（按分辨率与画质）| 中等（约 45 秒） | 5 | 最强准确度与美学、文字渲染 |
| **Nanobanana Pro** | 10 | 快（约 45 秒） | 5 | 高质量、复杂场景 |

## 图片模型

### GPT Image 2.0

| | |
| --- | --- |
| **Model ID** | `gpt-image-2` |
| **供应商** | OpenAI |
| **积分** | 按分辨率 × 画质，见下表 |
| **提示词长度** | 无硬性限制（建议常规长度） |
| **参考图片** | 最多 5 张 |
| **分辨率** | 1K, 2K, 4K |
| **画质** | Standard / Medium / High |
| **宽高比** | 1:1, 3:2, 2:3, 16:9, 9:16, 4:3, 3:4, 5:4, 4:5, 21:9, 9:21 |

**积分矩阵**：

| | Standard | Medium | High |
| --- | --- | --- | --- |
| **1K** | 2 | 10 | 18 |
| **2K** | 5 | 20 | 40 |
| **4K** | 10 | 32 | 80 |

免费用户可用 1K、2K；4K 需付费解锁。

当前默认模型，也是 MeiGen 目前**准确度和美学综合最强**的图片模型，特别擅长提示词遵循、文字渲染和复杂场景组合。

<Warning>
  **内容审核较严**：包含真实人物、名人、品牌 Logo 或暴力/成人元素时可能被拒。可尝试去掉具体姓名、品牌名或使用其他模型。
</Warning>

| **积分** | 每张 5 积分 |

</Warning>

### Midjourney Niji 7

| | |
| --- | --- |
| **Model ID** | `midjourney-niji7` |
| **供应商** | Midjourney |
| **积分** | 每张 15 积分 |
| **提示词长度** | 1-8,192 字符 |
| **参考图片** | 1 张（内容或风格参考） |
| **分辨率** | — |
| **宽高比** | 1:1, 4:3, 3:2, 16:9, 3:4, 2:3, 9:16 |

Midjourney 专注于动漫和插画的模型。能生成最高质量的艺术插画。支持 stylize、chaos、weird 和 style reference 等高级参数。

**独有功能：**
- 内容和风格参考模式
- 自动将非英文提示词翻译为英文
- 高级 Niji 7 参数（stylize、chaos、weird、raw 等）
- 为 Midjourney 语法优化的提示词[扩展模式](/zh/features/generate#提示词增强)

**权衡：**
- 最贵的图片模型（15 积分）
- 仅支持 1 张参考图片
- 生成速度较慢（见上方对比表）
- 每次生成返回 4 张候选图片（其他模型返回 1 张），可在图片详情中查看全部

### Midjourney V7

| | |
| --- | --- |
| **Model ID** | `midjourney-v7` |
| **供应商** | Midjourney |
| **积分** | 每张 15 积分 |
| **提示词长度** | 1-8,192 字符 |
| **参考图片** | 1 张（内容或风格参考） |
| **分辨率** | — |
| **宽高比** | 1:1, 4:3, 3:2, 16:9, 3:4, 2:3, 9:16 |

Midjourney 通用写实模型。与 Niji 7 共享相同的高级参数和参考图功能，但面向通用美学画面而非动漫风格。适合写实摄影、产品图、风景、人像等场景。

**与 Niji 7 相同：**
- 内容和风格参考模式
- 高级参数（stylize、chaos、weird、raw 等）
- 自动翻译非英文提示词
- 每次生成返回 4 张候选图片
- 仅支持 1 张参考图片

**与 Niji 7 不同：**
- 默认 stylize 为 0（Niji 7 默认 100），输出更贴近提示词本身
- 不自动追加动漫风格关键词
- 提示词增强使用通用润色模式（非 Niji 7 专用的动漫扩写）

## 视频模型

### Seedance 2.0

| | |
| --- | --- |
| **Model ID** | `seedance-2-0` |
| **供应商** | ByteDance |
| **积分** | 按秒计费（详见下方）|
| **视频时长** | 4-15 秒（默认 5 秒） |
| **参考图片** | 首帧/末帧（图生视频）|
| **参考视频** | 支持（视频续写） |
| **分辨率** | 480p, 720p, 1080p（默认 480p）|
| **宽高比** | adaptive, 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 |
| **音频** | 自动生成 |

ByteDance 最新视频生成模型。支持文生视频、图生视频（首帧/末帧）和视频续写三种模式。所有视频自带 AI 生成音频。`adaptive` 比例会根据参考图/参考视频自动适配尺寸。

#### 计费方式

按秒计费，分两档场景：

**无参考视频（文生视频 / 图生视频）**：

| 分辨率 | 单价 |
| --- | --- |
| 480p | 13 积分/秒 |
| 720p | 20 积分/秒 |
| 1080p | 22 积分/秒 |

**有参考视频（视频续写）**：单价更低，但有最低计费秒数保护。

| 分辨率 | 单价 |
| --- | --- |
| 480p | 8 积分/秒 |
| 720p | 14 积分/秒 |
| 1080p | 14 积分/秒 |

续写场景计费秒数 = `max(参考视频时长 + 你设的 duration, 最低计费秒数)`，对应表：

| 你设的 duration（秒） | 最低计费秒数 |
| --- | --- |
| 4 | 7 |
| 5 | 9 |
| 6 | 10 |
| 7 | 12 |
| 8 | 14 |
| 9 | 15 |
| 10 | 17 |
| 11 | 19 |
| 12 | 20 |
| 13 | 22 |
| 14 | 24 |
| 15 | 25 |

#### 计费示例

| 场景 | 计算 | 总积分 |
| --- | --- | --- |
| 480p × 5 秒文生视频 | 13 × 5 | **65** |
| 720p × 5 秒文生视频 | 20 × 5 | **100** |
| 720p × 10 秒文生视频 | 20 × 10 | **200** |
| 1080p × 10 秒文生视频 | 22 × 10 | **220** |
| 480p × 5 秒续写一段 3 秒视频 | 8 × max(3+5, 9) = 8 × 9 | **72** |
| 720p × 10 秒续写一段 5 秒视频 | 14 × max(5+10, 17) = 14 × 17 | **238** |

<Note>
  生成失败会自动退还**全额**积分，包括按秒计费的所有积分。
</Note>

### Veo 3.1

| | |
| --- | --- |
| **Model ID** | `veo-3.1` |
| **供应商** | Google |
| **积分** | 每个视频 20 积分 |
| **视频时长** | 8 秒（固定）|
| **参考图片** | 最多 2 张（作为首帧/末帧） |
| **分辨率** | 720p |
| **宽高比** | 16:9, 9:16 |
| **音频** | 有（生成音频） |

Google 的视频生成模型。创建带有 AI 生成音频的 8 秒短视频片段。支持速度控制（自动 / 慢速 / 快速）和风格预设（自动 / 写实 / 动漫 / 电影）。**适合需要稳定音画 + 风格预设**的场景；想要更长时长或更多比例选项请用 Seedance 2.0。

## 已知限制

<AccordionGroup>
  <Accordion title="Nanobanana 2 — 更严格的内容过滤" icon="triangle-exclamation">
    此模型对涉及真实人物、名人、品牌 Logo 和换脸请求的内容更加敏感。在其他模型上能用的提示词可能会被拒绝。如果你收到内容违规错误，重新措辞你的提示词以移除特定名称，或尝试其他模型。
  </Accordion>

  <Accordion title="Z Image Turbo — 不支持参考图片" icon="triangle-exclamation">
    此模型完全不支持参考图片。选择 Z Image Turbo 时，参考图片上传区域会被隐藏。如果需要使用参考图片，请切换到其他模型。
  </Accordion>

  <Accordion title="Niji 7 / Midjourney V7 — 仅支持单张参考图片" icon="triangle-exclamation">
    与支持最多 5 张参考图片的其他模型不同，Midjourney 系列模型仅接受 1 张参考图片。不过，它提供了更多控制参考图片使用方式的选项（内容参考 vs 风格参考）。
  </Accordion>

  <Accordion title="Niji 7 / Midjourney V7 — 每次返回 4 张候选图" icon="info-circle">
    与其他模型返回单张图片不同，Midjourney 系列模型每次生成会返回 4 张候选图片。所有候选图都会保存，可在图片详情对话框中浏览。4 张图片只收取一次费用（15 积分）。
  </Accordion>

  <Accordion title="GPT Image 2.0 — 内容审核较严" icon="triangle-exclamation">
    GPT Image 2.0 对真实人物、名人、品牌 Logo 和成人/暴力内容的审核比其他模型严格。可尝试去掉具体姓名、品牌名或使用其他模型。
  </Accordion>

  <Accordion title="Seedream — 2,000 字符提示词限制" icon="triangle-exclamation">
    两个 Seedream 模型的提示词上限为 2,000 字符，比 Gemini 模型的 4,000 字符限制要短。对于非常详细的提示词请做好规划。
  </Accordion>

  <Accordion title="Seedance 2.0 — 按秒计费 + 续写最低计费" icon="circle-info">
    与图片模型按张固定收费不同，Seedance 按秒计费：480p 为 13 积分/秒，720p 为 20 积分/秒，1080p 为 22 积分/秒。提交前侧边栏会显示精确的总积分。

    **续写视频时**：单价虽然降低（480p 8 积分/秒、720p 14 积分/秒、1080p 14 积分/秒），但有最低计费秒数保护（防止短续写被参考视频长度拉低单价）。例如续写 5 秒时最低按 9 秒计费，续写 15 秒时最低按 25 秒计费。详细对照表见上方 Seedance 2.0 章节。
  </Accordion>

  <Accordion title="Veo 3.1 — 时长固定 8 秒" icon="triangle-exclamation">
    Veo 3.1 每次生成固定 8 秒视频，无法调节时长。仅支持 16:9 / 9:16 两种比例和 720p 分辨率。如需更长时长、更多比例或 480p 选项，请使用 Seedance 2.0。
  </Accordion>
</AccordionGroup>

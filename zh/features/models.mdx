MeiGen 支持多种 AI 模型用于图片和视频生成。每种模型有不同的优势、费用和限制。本页帮助你根据使用场景选择合适的模型。

## 图片模型

### Nanobanana 2

| | |
| --- | --- |
| **供应商** | Google (Gemini) |
| **积分** | 每张 5 积分 |
| **提示词长度** | 5-4,000 字符 |
| **参考图片** | 最多 5 张 |
| **分辨率** | 2K, 4K |
| **宽高比** | 1:1, 3:4, 4:5, 2:3, 9:16, 1:4, 1:8, 4:3, 5:4, 3:2, 16:9, 21:9, 4:1, 8:1 |

最实惠的 Gemini 模型，支持最广泛的宽高比，包括超宽（8:1）和超高（1:8）选项。适合快速迭代和探索。

<Warning>
  **内容审核**：Nanobanana 2 的内容过滤比其他模型更严格。涉及**名人、品牌名称、真实人物或换脸场景**的提示词更容易被拒绝。如果你的提示词被标记，尝试改写以避免使用特定名称，或使用其他模型。
</Warning>

### Midjourney Niji 7

| | |
| --- | --- |
| **供应商** | Midjourney |
| **积分** | 每张 15 积分 |
| **提示词长度** | 1-8,192 字符 |
| **参考图片** | 1 张（内容或风格参考） |
| **分辨率** | — |
| **宽高比** | 1:1, 4:3, 3:2, 16:9, 3:4, 2:3, 9:16 |

Midjourney 专注于动漫和插画的模型。能生成最高质量的艺术插画。支持 stylize、chaos、weird 和 style reference 等高级参数。

**独有功能：**
- 内容和风格参考模式
- 自动将非英文提示词翻译为英文
- 高级 Niji 7 参数（stylize、chaos、weird、raw 等）
- 为 Midjourney 语法优化的提示词[扩展模式](/zh/features/prompt-enhance)

**权衡：**
- 最贵的图片模型（15 积分）
- 仅支持 1 张参考图片
- 生成速度较慢（约 60 秒）
- 每次生成返回 4 张候选图片（其他模型返回 1 张），可在图片详情中查看全部

### Z Image Turbo

| | |
| --- | --- |
| **供应商** | Alibaba (Qwen) |
| **积分** | 每张 2 积分 |
| **提示词长度** | 5-4,000 字符 |
| **参考图片** | 不支持 |
| **分辨率** | 固定 |
| **宽高比** | 1:1, 2:3, 3:2, 3:4, 4:3, 7:9, 9:7, 9:16, 9:21, 16:9, 21:9 |

最快、最便宜的模型。非常适合快速原型制作和需要秒级出结果的快速迭代。

<Warning>
  **不支持参考图片。** Z Image Turbo 仅从文本生成图片 — 不能上传参考图片。如果需要使用参考图片，请选择其他模型。
</Warning>

### GPT image1.5

| | |
| --- | --- |
| **供应商** | RunningHub |
| **积分** | 每张 2 积分 |
| **提示词长度** | 5-800 字符 |
| **参考图片** | 最多 2 张 |
| **分辨率** | — |

**限制：**

- 最多 2 张参考图片

## 视频模型

### Veo 3.1

| | |
| --- | --- |
| **供应商** | Google |
| **积分** | 每个视频 20 积分 |
| **参考图片** | 最多 2 张（作为首帧/末帧） |
| **分辨率** | 720p |
| **宽高比** | 16:9, 9:16 |
| **音频** | 有（生成音频） |

Google 的视频生成模型。创建带有 AI 生成音频的短视频片段。支持速度控制（自动 / 慢速 / 快速）和风格预设（自动 / 写实 / 动漫 / 电影）。

## 已知限制

<AccordionGroup>
  <Accordion title="Nanobanana 2 — 更严格的内容过滤" icon="triangle-exclamation">
    此模型对涉及真实人物、名人、品牌 Logo 和换脸请求的内容更加敏感。在其他模型上能用的提示词可能会被拒绝。如果你收到内容违规错误，重新措辞你的提示词以移除特定名称，或尝试其他模型。
  </Accordion>

  <Accordion title="Z Image Turbo — 不支持参考图片" icon="triangle-exclamation">
    此模型完全不支持参考图片。选择 Z Image Turbo 时，参考图片上传区域会被隐藏。如果需要使用参考图片，请切换到其他模型。
  </Accordion>

  <Accordion title="Niji 7 — 仅支持单张参考图片" icon="triangle-exclamation">
    与支持最多 5 张参考图片的其他模型不同，Niji 7 仅接受 1 张参考图片。不过，它提供了更多控制参考图片使用方式的选项（内容参考 vs 风格参考）。
  </Accordion>

  <Accordion title="Niji 7 — 每次返回 4 张候选图" icon="info-circle">
    与其他模型返回单张图片不同，Niji 7 每次生成会返回 4 张候选图片。所有候选图都会保存，可在图片详情对话框中浏览。4 张图片只收取一次费用（15 积分）。
  </Accordion>

  <Accordion title="GPT image1.5 — 比例和提示词长度有限" icon="triangle-exclamation">
    仅支持 3 种宽高比（1:1、3:2、2:3），提示词限制为 800 字符。使用此模型时请保持提示词简洁。
  </Accordion>

  <Accordion title="Seedream — 2,000 字符提示词限制" icon="triangle-exclamation">
    两个 Seedream 模型的提示词上限为 2,000 字符，比 Gemini 模型的 4,000 字符限制要短。对于非常详细的提示词请做好规划。
  </Accordion>
</AccordionGroup>

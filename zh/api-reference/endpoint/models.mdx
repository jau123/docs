---
title: "模型列表"
description: "获取可用的 AI 模型及其配置"
---

```
GET /api/models
```

返回所有可用于生成的活跃 AI 模型列表。这是一个公开端点 — 无需认证。

响应缓存 5 分钟。

## 响应

返回包含 `models` 数组的对象：

<ResponseField name="id" type="string">
  模型的唯一标识符。生成时用作 `modelId`。
</ResponseField>

<ResponseField name="name" type="string">
  模型的显示名称。
</ResponseField>

<ResponseField name="provider" type="string">
  模型供应商（如 "Google"、"ByteDance"、"Midjourney"）。
</ResponseField>

<ResponseField name="credits_per_generation" type="number">
  每次生成消耗的基础积分。
</ResponseField>

<ResponseField name="supported_ratios" type="string[]">
  支持的宽高比数组（如 `["1:1", "3:4", "16:9"]`）。
</ResponseField>

<ResponseField name="max_reference_images" type="number">
  接受的最大参考图片数量（0 = 不支持）。
</ResponseField>

<ResponseField name="media_type" type="string">
  输出类型：`"image"` 或 `"video"`。
</ResponseField>

<ResponseField name="extra_config" type="object">
  额外的模型特定配置（分辨率、定价层级、标签等）。
</ResponseField>

The MeiGen gallery is a curated collection of AI-generated images from the community. You can browse for inspiration, save favorites, and use any image as a starting point for your own creations.

## Browsing

The gallery displays images in a responsive masonry grid layout. On desktop, hovering over a card reveals the overlay with actions and metadata.

Each card shows:

- **Author info** — avatar, display name, and a link to their profile
- **Stats** — likes, views, and time posted
- **Model badge** — which AI model was used to create the image

## Card Actions

When you hover over a gallery card (desktop) or tap it (mobile), you get access to several actions:

<AccordionGroup>
  <Accordion title="Use Idea (Recreate)" icon="repeat">
    Appears when the image has a usable prompt. Clicking this copies the original prompt into the generation sidebar, letting you recreate the image or use it as a starting point for variations.

    The button only shows on cards where `promptReady` is true — meaning the original prompt has been extracted and is available.
  </Accordion>

  <Accordion title="Favorite" icon="heart">
    Save the image to your favorites collection. Click again to remove it. Requires sign-in.

    Access all your saved images from the **Favorites** page in the left sidebar.
  </Accordion>

  <Accordion title="View on X" icon="arrow-up-right-from-square">
    Opens the original tweet on X (Twitter) in a new tab, so you can see the full context and engage with the author.
  </Accordion>
</AccordionGroup>

## Search

The search panel provides two tabs:

### Posts

Search across the entire gallery using keywords. The search uses a hybrid approach combining text similarity and vector embeddings for accurate results. You can filter by:

- **Model** — filter by specific AI model
- **Category** — browse by content category
- **Has prompt** — only show images with available prompts

### Generations

Search your own generated images (requires sign-in). Filter by prompt text, model, or date range.

## Favorites

Save images you like to your personal collection. Favorites are synced across devices and support two content types:

- **Gallery posts** — curated images from the community
- **Your generations** — images you've generated yourself

Access your favorites from the sidebar or navigate to the Favorites page. You can filter by type (All / Posts / Generations).

## Image Detail

Click any image to open the detail dialog, which shows:

- Full-resolution image
- Original prompt (if available)
- Model and generation parameters
- Author information and stats

From the detail view, you can:

- **Copy Prompt** — copy the text to your clipboard
- **Use as Reference** — add the image as a reference for a new generation
- **Download** — save the image to your device
- **Reverse Prompt** — use AI to analyze the image and generate a descriptive prompt
- **Share** — share the image on X (Twitter)

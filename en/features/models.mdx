| --- | --- |

| **Prompt length** | No hard limit (standard prompts recommended) |

**Credit matrix:**

| | Standard | Medium | High |
| --- | --- | --- | --- |
| **1K** | 2 | 10 | 50 |
| **2K** | 5 | 25 | 100 |

Free users can pick 1K and 2K; 4K requires a paid plan.

The current default model and MeiGen's **best overall image model** for prompt accuracy and aesthetic quality. Excels at instruction following, in-image text rendering, and complex multi-element scenes.

<Warning>
  **Stricter moderation**: prompts involving real people, celebrities, brand logos, or violent/adult content may be rejected. Rephrase to drop specific names or try a different model.
</Warning>

### Midjourney Niji 7

| | |
| --- | --- |
| **Model ID** | `midjourney-niji7` |
| **Provider** | Midjourney |
| **Credits** | 15 per image |
| **Prompt length** | 1–8,192 characters |
| **Reference images** | 1 (content or style reference) |
| **Resolutions** | — |
| **Aspect ratios** | 1:1, 4:3, 3:2, 16:9, 3:4, 2:3, 9:16 |

Midjourney's anime and illustration-focused model. Produces the highest quality artistic illustrations. Supports advanced parameters like stylize, chaos, weird, and style references.

**Unique features:**
- Content and style reference modes
- Auto-translates non-English prompts to English
- Advanced Niji 7 parameters (stylize, chaos, weird, raw, etc.)
- Prompt [Expand mode](/en/features/generate#prompt-enhancement) optimized for Midjourney syntax

**Trade-offs:**
- Most expensive image model (15 credits)
- Only 1 reference image supported
- Slower generation (see the comparison table above)
- Returns 4 candidate images per generation (other models return 1) — all viewable in the image detail dialog

| **Provider** | Midjourney |

Midjourney's general-purpose photorealistic model. Shares the same advanced parameters and reference image features as Niji 7, but targets universal aesthetics rather than anime style. Ideal for realistic photography, product shots, landscapes, portraits, and more.

**Same as Niji 7:**
- Content and style reference modes
- Advanced parameters (stylize, chaos, weird, raw, etc.)
- Auto-translates non-English prompts to English
- Returns 4 candidate images per generation

**Different from Niji 7:**
- Default stylize is 0 (Niji 7 defaults to 100) — output stays closer to your prompt
- Does not auto-append anime style keywords
- Prompt enhance uses general polish mode (not Niji 7's anime-specific expand)

## Video Models

### Seedance 2.0

| | |
| --- | --- |
| **Model ID** | `seedance-2-0` |
| **Provider** | ByteDance |
| **Credits** | Per-second pricing (see below) |
| **Duration** | 4-15 seconds (default 5) |
| **Reference images** | First/last frame (image-to-video) |
| **Reference video** | Yes (video continuation) |
| **Resolutions** | 480p, 720p (default 480p) |
| **Aspect ratios** | adaptive, 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 |
| **Audio** | Auto-generated |

ByteDance's latest video generation model. Supports text-to-video, image-to-video (first/last frame), and video continuation. All outputs include AI-generated audio. The `adaptive` aspect ratio automatically matches the reference image/video dimensions.

#### Pricing

Per-second pricing, in two tiers:

**Without reference video (text-to-video / image-to-video):**

| Resolution | Rate |
| --- | --- |
| 480p | 13 credits/sec |
| 720p | 26 credits/sec |

**With reference video (video continuation):** lower rate, but with a minimum-billable-seconds floor.

| Resolution | Rate |
| --- | --- |
| 480p | 8 credits/sec |
| 720p | 16 credits/sec |

For continuation, billable seconds = `max(reference video duration + your duration, minimum billable)`. Lookup table:

| Your duration (sec) | Min billable (sec) |
| --- | --- |
| 4 | 7 |
| 5 | 9 |
| 6 | 10 |
| 7 | 12 |
| 8 | 14 |
| 9 | 15 |
| 10 | 17 |
| 11 | 19 |
| 12 | 20 |
| 13 | 22 |
| 14 | 24 |
| 15 | 25 |

#### Pricing Examples

| Scenario | Calculation | Total |
| --- | --- | --- |
| 480p × 5s text-to-video | 13 × 5 | **65** |
| 720p × 5s text-to-video | 26 × 5 | **130** |
| 720p × 10s text-to-video | 26 × 10 | **260** |
| 480p × 5s continuing a 3s video | 8 × max(3+5, 9) = 8 × 9 | **72** |
| 720p × 10s continuing a 5s video | 16 × max(5+10, 17) = 16 × 17 | **272** |

<Note>
  If generation fails, the **full** credit amount is automatically refunded — including all per-second charges.
</Note>

## Known Limitations

<AccordionGroup>
  <Accordion title="Nanobanana 2 — Stricter content filtering" icon="triangle-exclamation">
    This model is more sensitive to content involving real people, celebrities, brand logos, and face-swap requests. Prompts that work on other models may be rejected here. If you get a content violation error, rephrase your prompt to remove specific names or try a different model.
  </Accordion>

  <Accordion title="Z Image Turbo — No reference images" icon="triangle-exclamation">
    This model does not support reference images at all. The reference image upload section will be hidden when Z Image Turbo is selected. If you need to use reference images, switch to any other model.
  </Accordion>

  <Accordion title="Niji 7 / Midjourney V7 — Single reference image only" icon="triangle-exclamation">
    Unlike other models that support up to 5 reference images, Midjourney models only accept 1 reference image. However, they offer more control over how the reference is used (content vs. style reference).
  </Accordion>

  <Accordion title="Niji 7 / Midjourney V7 — Returns 4 candidate images" icon="info-circle">
    Unlike other models that return a single image, Midjourney models return 4 candidate images per generation. All candidates are saved and can be browsed in the image detail dialog. You are only charged once (15 credits) for all 4 images.
  </Accordion>

  <Accordion title="GPT Image 2.0 — Stricter moderation" icon="triangle-exclamation">
    GPT Image 2.0 applies stricter moderation than other models to real people, celebrities, brand logos, and adult/violent content. Rephrase your prompt to drop specific names, or try a different model.
  </Accordion>

  <Accordion title="Seedream — 2,000 character prompt limit" icon="triangle-exclamation">
    Both Seedream models cap prompts at 2,000 characters, shorter than the 4,000-character limit of Gemini models. Plan accordingly for very detailed prompts.
  </Accordion>

  <Accordion title="Seedance 2.0 — Per-second pricing + continuation minimums" icon="circle-info">
    Unlike image models with flat per-image pricing, Seedance is billed per second: 13 credits/sec for 480p, 26 credits/sec for 720p. The sidebar shows the exact total before you submit.

    **For video continuation**: rates drop (480p 8/sec, 720p 16/sec), but a minimum-billable-seconds floor applies (so a short continuation isn't unfairly cheapened by a long reference video). For example, continuing 5 seconds bills at least 9 seconds; continuing 15 seconds bills at least 25 seconds. See the full lookup table in the Seedance 2.0 section above.
  </Accordion>

  <Accordion title="Veo 3.1 — Fixed 8-second duration" icon="triangle-exclamation">
    Veo 3.1 always generates 8-second videos — duration is not adjustable. Only supports 16:9 / 9:16 aspect ratios and 720p resolution. For longer durations, more ratios, or 480p output, use Seedance 2.0.
  </Accordion>
</AccordionGroup>

## Step 1: Create an Account

Go to [meigen.ai](https://www.meigen.ai) and sign in with your Google account.

New users receive free credits to get started. You can also purchase additional credits from the billing page.

## Step 2: Get an API Key

<Steps>
  <Step title="Open API Keys page">
    Click your avatar in the top-right corner, then select **API Keys** from the dropdown menu.
  </Step>
  <Step title="Create a new key">
    Click **Create New Key**, give it a name (e.g. "My App"), and click **Create**.
  </Step>
  <Step title="Copy the key">
    Copy the key immediately — it starts with `meigen_sk_` and won't be shown again.
  </Step>
</Steps>

<Warning>
  API calls can only use **purchased credits**, not daily free credits. Make sure your account has purchased credits before making API calls.
</Warning>

## Step 3: Generate Your First Image

<CodeGroup>
```bash cURL
curl -X POST https://www.meigen.ai/api/generate/v2 \
  -H "Authorization: Bearer meigen_sk_YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A serene Japanese garden with cherry blossoms, soft morning light",
    "modelId": "nanobanana-2",
    "aspectRatio": "16:9"
  }'
```

```javascript Node.js
const response = await fetch('https://www.meigen.ai/api/generate/v2', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer meigen_sk_YOUR_API_KEY',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    prompt: 'A serene Japanese garden with cherry blossoms, soft morning light',
    modelId: 'nanobanana-2',
    aspectRatio: '16:9',
  }),
});

const data = await response.json();
console.log('Generation ID:', data.generationId);
```

```python Python
import requests

response = requests.post(
    'https://www.meigen.ai/api/generate/v2',
    headers={
        'Authorization': 'Bearer meigen_sk_YOUR_API_KEY',
        'Content-Type': 'application/json',
    },
    json={
        'prompt': 'A serene Japanese garden with cherry blossoms, soft morning light',
        'modelId': 'nanobanana-2',
        'aspectRatio': '16:9',
    },
)

data = response.json()
print('Generation ID:', data['generationId'])
```
</CodeGroup>

The API returns a `generationId` immediately. The image is generated asynchronously — poll the status endpoint to check when it's ready.

## Step 4: Check Generation Status

```bash
curl https://www.meigen.ai/api/generate/v2/status/GENERATION_ID \
  -H "Authorization: Bearer meigen_sk_YOUR_API_KEY"
```

Response when complete:

```json
{
  "status": "completed",
  "imageUrl": "https://images.meigen.art/generations/xxx.png",
  "prompt": "A serene Japanese garden with cherry blossoms, soft morning light",
  "model": "nanobanana-2",
  "aspectRatio": "16:9",
  "createdAt": "2025-03-01T12:00:00Z"
}
```
